# Neon Void - Endless Runner

A fast-paced, neon-styled endless runner game built with Vanilla JavaScript, HTML, and CSS.

## Features
- **Addictive Gameplay**: Dodge obstacles that speed up over time.
- **High Score System**: Locally saves the top 10 scores using your browser's LocalStorage.
- **Responsive Design**: Works on desktop (arrow keys) and mobile (touch sides of screen).
- **Ad-Ready**: Dedicated spaces for top and bottom banner advertisements.

## How to Play
1. Open `index.html` in your web browser.
2. Press **Start Game**.
3. Use **Arrow Keys** (Left/Right) or tap the **Left/Right** side of the screen to move.
4. Avoid the falling neon blocks.
5. If you crash, and your score is high enough, enter your name to join the **Top 10 Legends**!

## Customization
- **Ads**: You can replace the `.ad-banner` divs in `index.html` with your actual ad network code (e.g., Google AdSense).
- **Difficulty**: Tweak `CONFIG` values in `script.js` to change speed, spawn rates, or colors.
